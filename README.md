# VritAssessment

## Features

- Next.js 15.3
- Tailwind CSS

- Framer Motion animations
- ESLint for linting

## Installation

To install dependencies and start the development server:

- Run `npm install`

## Usage

- Run `npm run dev` to start the development server.

## Dependencies

- next@15.3.5
- react@^19.0.0
- react-dom@^19.0.0
- framer-motion@^12.23.3
